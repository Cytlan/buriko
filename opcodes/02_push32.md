0x02 - push32
=============

Pushes an 32-bit value onto the stack. The value is signed.

Mnemonic  | Opcode | Other bytes | Stack
----------|--------|-------------|--------------
push32    | 02     | 4: value    | â†’ value

Operands
--------

#### Other bytes

* 1 - `int` - Value to push

#### Stack in

None

#### Stack out

* 1 - `int` - Value

Example
-------

```
	push32 1000000
```

-------------

Version | Note
--------|-------------

Equivalent C code
-----------------

```c
int op_push32(VMThread* thread)
{
	VM_PushStack(thread, VM_ReadPC32(thread));
	return 0;
}
```
